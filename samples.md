# Sample Recordings

Reference recordings from the [Singing Insects of North America (SINA)](https://orthsoc.org/sina/a00samples.htm) database, used for spectral analysis to derive SynthDef parameters.

## Species and Sources

| Species | Common Name | SINA Page | Audio File |
|---|---|---|---|
| *Gryllus pennsylvanicus* | Fall field cricket | [489a](https://orthsoc.org/sina/489a.htm) | [489ss2.wav](https://orthsoc.org/sina/media/489ss2.wav) |
| *Oecanthus fultoni* | Snowy tree cricket | [585a](https://orthsoc.org/sina/585a.htm) | [585ss2.wav](https://orthsoc.org/sina/media/585ss2.wav), [585sso.wav](https://orthsoc.org/sina/media/585sso.wav) |
| *Pterophylla camellifolia* | Common true katydid | [141a](https://orthsoc.org/sina/141a.htm) | [141ss1.wav](https://orthsoc.org/sina/media/141ss1.wav) |
| *Neoconocephalus robustus* | Robust conehead | [195a](https://orthsoc.org/sina/195a.htm) | [195ss.wav](https://orthsoc.org/sina/media/195ss.wav) |
| *Allonemobius allardi* | Allard's ground cricket | [523a](https://orthsoc.org/sina/s523a.htm) | [532ss2_4.wav](https://orthsoc.org/sina/media/532ss2_4.wav) |

## Usage

Download the files into a `samples/` directory, then run the analysis:

```bash
python analyze.py
```
